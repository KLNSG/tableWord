# tableWord

#### 介绍
从mysql数据库导数据生成数据库word文档

#### 软件架构
软件架构说明
springboot + mybatis

#### 安装教程

1. 修改application.properties里面数据库连接配置，配置生成的文件地址，数据库名
2. 访问: localhost:8888/test/demo
3. 数据库文档生成
